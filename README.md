## **요약**
![image](https://user-images.githubusercontent.com/72062916/226169371-9a89b3f4-d0a6-494a-86fc-5cf9e2a270ab.png)

면세점 사이트에서 가능한 상품 정보 확인, 주문, 결제, 주문 확인 등의 주요기능이 있습니다. </br>
그 외의 추기기능으로 리뷰, 다국어 처리, 관리자 페이지 등의 기능을 제공합니다.
</br>
